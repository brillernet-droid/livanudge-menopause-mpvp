# LivaNudge 更年期女性试点最小可行产品说明

## 产品定位

LivaNudge 更年期版本是一个面向 45-65 岁围绝经期及绝经后早期女性的生活方式支持工具。它不替代医生，不提供诊断、处方、用药或荷尔蒙治疗建议，而是帮助用户每天记录身体信号，并把症状困扰转化成低负担、可完成、可追踪的小行动。

## 目标用户

- 正在经历月经变化、潮热、夜汗、睡眠下降、情绪波动或疲劳的女性。
- 绝经后早期，开始关注骨骼健康、力量、体重、盆底和长期健康风险的女性。
- 愿意使用手机完成短记录，但不希望被当作病人管理的女性。
- 香港试点场景下，可由伴侣、成年子女、社区服务人员或研究团队提供支持。

## 试点闭环

1. 用户完成每日短记录：睡眠、潮热、夜汗、心情、压力、精力、月经状态和身体困扰。
2. 系统先检查安全提示：绝经后出血、大量出血、胸痛、明显气促、神经症状、自伤想法。
3. 没有安全提示时，返回 1-3 个舒缓小行动。
4. 用户完成后形成一周趋势，用于自我理解和试点评估。
5. 高困扰或安全提示进入服务团队跟进队列。

## 首版功能范围

- 手机优先界面。
- 繁体中文、简体中文、英文三套完整语言。
- 每日症状记录。
- 今日舒缓小行动卡片和图片占位。
- 安全提示固定流程。
- 支持者授权与协助记录。
- 试点工作台。
- 模拟接口和未来后端接入入口。
- 去识别化资料导出。

## 先不做

- 医疗诊断。
- 药物、补充剂或荷尔蒙治疗建议。
- 紧急情况判断。
- 真实医生预约。
- 真实支付。
- 复杂可穿戴设备接入。
